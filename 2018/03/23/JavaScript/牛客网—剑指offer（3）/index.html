2.链表中倒数第k个结点

题目描述
输入一个链表，输出该链表中倒数第k个结点。

思路
两个指针，先让第一个指针和第二个指针都指向头结点，首先让第一个指针走(k-1)步，到达第k个节点。然后两个指针同时往后移动，当第一个结点到达末尾的时候，第二个结点所在位置就是倒数第k个节点了，如果k最终的值大于等于1，则k的值已经超出链表的长度了
这里没说清楚那就当第1个节点为1吧(k>=1)
